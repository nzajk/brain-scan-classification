# Brain Scan Classification
## Overview
This repository contains the code to a full-stack application which allows the user to input brain scans and receive brain tumor diagnostics.

## Dataset
The dataset used to train the machine learning model can be found [here](https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset/data).

## Installation
1. Clone the repository
   ```
   git clone https://github.com/nzajk/brain-scan-classification.git
   ```
2. Install Requirements
   ```
   pip install -r requirements.txt
   ```
3. Host the API
   ```
   python src/backend/app.py
   ```
3. Open the Frontend
   - Navigate to `src/frontend/index.html`
   - Open it in your browser of choice

# Brain Scan Classification
## Description 
Within this repository holds a machine learning model which will be able to classify brain scans into the following categories, healthy vs. diseased. This model should be used with MRI images and each image can easily be tested using the make_prediction function that I have provided near the end of the .ipynb file. 

## Getting Started

### Dependencies
To download all necessary libraries copy paste the following into your terminal (use pip3 for Python3).
```
pip install tensorflow
pip install opencv-python
pip install scikit-learn
pip install numpy
pip install matplotlib
pip install glob2
```

### Setup
After installing all necessary packages simply just clone the repository and use the .ipynb file to make any predictions.
